## Overview

Commands are defined in Markdown files with specific sections.

## Location

```
plugins/your-plugin/commands/your-command.md
```

## Template

```markdown
# Your Command

Brief description.

## Usage

/your-command [options]

## Purpose

When and why to use this command.

## Instructions

1. Step-by-step instructions
2. What the command should do
3. Expected outcomes

## Examples

### Example 1

/your-command
```
