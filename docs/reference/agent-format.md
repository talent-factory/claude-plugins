## Overview

Agents are specialized AI assistants defined in Markdown files.

## Location

```
plugins/your-plugin/agents/your-agent.md
```

## Template

```markdown
# Your Agent

## Metadata

- **Name**: your-agent
- **Description**: Brief description
- **Version**: 1.0.0
- **Tags**: tag1, tag2

## Expertise

Areas of expertise.

## Approach

Methodology and approach.

## Examples

Example interactions.
```
