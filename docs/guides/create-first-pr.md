## Quick Overview

The `/create-pr` command helps you create professional pull requests with:

- **Automated PR description** - Generated from commits
- **Template support** - Uses repository PR templates
- **Best practices** - Follows PR conventions
- **Integration** - Works with GitHub, GitLab, Bitbucket
